Project Overview

This project is an AI-powered American Sign Language (ASL) prediction system that translates hand gestures into readable text, facilitating communication between ASL speakers and non-ASL speakers. The core of this system is a deep learning model based on VGG16 architecture, which is fine-tuned to recognize ASL gestures.

Objective

The main objective is to build a user-friendly tool that bridges communication gaps by accurately identifying ASL gestures, helping create a more inclusive environment for those who rely on sign language.

Key Features

  • Real-time ASL gesture recognition with high accuracy using VGG16.
  • Image upload functionality integrated with a web interface built using Flask.
  • Simple and responsive UI for users to navigate and get predictions easily.
  • Optimized for quick response times and seamless communication support.
  • Adaptable for adding new gestures and expanding the vocabulary in future updates.

Technologies Used

  • Python & TensorFlow/Keras: For building and fine-tuning the VGG16-based deep learning model tailored for ASL gesture recognition.
  • Flask: Used as the backend framework to handle image uploads and API requests for predictions.
  • HTML, CSS: For creating an interactive and user-friendly front-end interface.
  • OpenCV: Utilized for preprocessing images before passing them to the model for accurate predictions.

Future Enhancements

Planned future improvements include:

  • Adding more complex ASL phrases and gestures.
  • Integrating audio output for translating gestures to spoken words.
  • Creating a mobile app version for on-the-go use.
  • Using continuous learning to improve the model's accuracy as more data is added.
